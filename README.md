# 🎮 Host Gamer - Landing Page

## 📌 Sobre o Projeto

O **Host Gamer** é um projeto de estudo desenvolvido para praticar conceitos de **desenvolvimento Front-End**, criação de interfaces modernas e construção de uma landing page profissional.

A proposta do projeto foi criar um site fictício para uma empresa de hospedagem gamer, oferecendo planos de servidores, suporte com inteligência artificial e uma experiência visual inspirada no universo dos jogos.

O foco principal foi desenvolver uma interface:

- Moderna
- Responsiva
- Visualmente atrativa
- Com identidade gamer
- Fácil de navegar

---

# 🎯 Objetivo do Projeto

O objetivo foi colocar em prática conhecimentos de:

- Estruturação HTML5
- Estilização avançada com CSS3
- Organização de layouts
- Criação de componentes visuais
- Experiência do usuário (UX/UI)
- Responsividade
- Desenvolvimento de uma página comercial

# 🎨 Processo de Criação

## 1. Planejamento Inicial

Antes de iniciar o desenvolvimento foi definido:

### Tema:

> Empresa fictícia de hospedagem de servidores gamer.

### Público-alvo:

- Jogadores
- Comunidades online

---

# 🖥️ Estrutura do Site

O site foi dividido em quatro principais áreas:

---

# 🏠 1. Página Inicial (Home)

A primeira seção foi criada para causar impacto visual.

Elementos utilizados:

- Background gamer escuro
- Personagens ao fundo
- Logo da empresa
- Menu de navegação
- Texto principal
- Botão de chamada para ação


Objetivo:

Apresentar rapidamente a proposta da empresa e incentivar o usuário a conhecer os serviços.

---

# 👥 2. Seção Sobre Nós

Criada para apresentar a história da empresa.

Elementos:

- Imagem ilustrativa da marca
- Texto institucional
- Missão e valores


O objetivo foi transmitir:

- Confiança
- Profissionalismo
- Experiência no mercado gamer

---

# 💎 3. Seção de Planos

Foi desenvolvida uma área de preços utilizando cards individuais.

Planos criados:

## Single

Indicado para usuários iniciantes.

Recursos:

- 1 site
- 10GB SSD
- Email profissional


---

## Premium

Plano intermediário.

Recursos:

- Até 3 sites
- 20GB SSD
- Mais recursos de hospedagem


---

## Business

Plano avançado.

Recursos:

- Até 50 sites
- 50GB NVMe
- Maior desempenho


Cada card possui:

- Nome do plano
- Desconto
- Preço antigo
- Preço atual
- Lista de benefícios
- Botão de contratação

---

# 🤖 4. Seção de Assistente IA

Foi criada uma área apresentando um assistente virtual chamado:

> Gehost

Objetivo:

Auxiliar o usuário na escolha do plano ideal.

Elementos:

- Título destacado
- Descrição
- Botão de contato

# ⚙️ Desenvolvimento

## Construção do HTML

Primeiramente foi criada a estrutura sem estilos.

Foram adicionados:

- Header
- Navbar
- Sections
- Cards
- Botões


Depois foram adicionadas classes para facilitar a estilização.

Exemplo:

```html
<section class="planos">

<div class="card">

<h2>Premium</h2>

<p>
Tudo que você precisa para começar
</p>

</div>

</section>
```

---

# 🎨 Desenvolvimento do CSS

Após a estrutura HTML pronta, iniciou-se a estilização.

Principais recursos utilizados:

## Backgrounds

Aplicação de imagens e gradientes:

```css
body{

background:#000;

}
```


---

## Cards de Planos

Foram criados cards com:

- Bordas arredondadas
- Sombras
- Espaçamento interno
- Destaques de preço


Exemplo:

```css
.card{

border-radius:30px;

padding:40px;

background:white;

}
```

---

# 📱 Responsividade

O projeto foi pensado para funcionar em diferentes dispositivos.

Foram utilizados:

- Flexbox
- Media Queries
- Unidades relativas


Exemplo:

```css
@media(max-width:768px){

.planos{

flex-direction:column;

}

}
```
